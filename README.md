# João Louro — Portfolio Website

A clean, minimal personal portfolio with a **liquid glass** UI aesthetic, built with HTML, CSS, and JavaScript.

> *"Build things worth remembering."*

## Pages

- **Home** — Hero section with introduction, tech stack and creations
- **About** — Who I am, education and interests
- **Contact** — Contact form powered by Formspree

## Features

- **Liquid glass UI** — backdrop blur, translucent panels and subtle glow effects
- **Dark theme** — deep dark background with cyan/blue accent
- **Responsive** — optimised for desktop and mobile
- **Contact form** — Formspree integration with no-redirect fetch submission
- **Sticky header** — floating glass pill navbar

## Tech Stack

- HTML5
- CSS3 (custom properties, grid, flexbox)
- JavaScript (vanilla)
